Liquid Glass

A buildless, zero-dependency engine that gives the web Apple-style refractive glass: edges that bend the background, a faint color fringe, frost, and a living rim.

This one is a desktop thing

The refraction is powered by an SVG filter inside backdrop-filter, and only Chromium browsers (Chrome, Edge, Brave) actually render it. Every browser on iPhone and iPad runs on Apple’s WebKit engine, which does not, so there is no way to show the real effect on a phone yet.

It is not broken and it is not your browser. The web platform simply cannot do this on iOS today. So instead of a half-working page, here is what it looks like on a computer.

Panels bending the background at the rim, with chromatic aberration.

Liquid Glass

How it works — 5 layers

The effect is just five honest CSS/SVG layers stacked in order. Here's each one, peeled apart, with the exact code responsible.

Layer 01 · Tint

A translucent fill

Start with a barely-there background color. It sets the mood — clear, frosted, dark, or tinted — before anything bends light.

.glass { /* --glass-tint */
  background: rgba(255,255,255,.06);
}
tint only
Layer 02 · Blur & saturate

Frost the backdrop

A backdrop-filter blurs and over-saturates whatever is behind the pane — the classic "frosted" legibility trick, and the Safari/Firefox fallback when SVG refraction isn't available.

.glass {
  backdrop-filter: blur(6px) saturate(170%);
}
blur + saturate
Layer 03 · Specular rim

The wet 3D bevel

Inset box-shadows fake a lit top edge and a shadowed bottom — the "wet glass" highlight — plus an outer shadow so the pane floats.

.glass {
  box-shadow:
    inset 0 1px 1px rgba(255,255,255,.55), /* top light */
    inset 0 -1px 1px rgba(255,255,255,.18),/* bottom */
    0 6px 24px rgba(0,0,0,.22);            /* float */
}
bevel + shadow
Layer 04 · Moving shine

A travelling highlight

A diagonal gradient pseudo-element gives a fixed sheen, and a radial highlight follows the cursor — set --mx/--my on hover for the "alive" feel.

.glass::after {
  background: radial-gradient(130px circle
    at var(--mx) var(--my), rgba(255,255,255,.28), transparent);
}
hover me →
Layer 05 · SVG refraction

Edge bending & chromatic aberration

The signature move: a per-element SVG displacement map pushes the backdrop pixels outward at the rounded rim, and the red/blue channels are displaced by slightly different amounts to fringe the edge — exactly like a real lens. This is what createLiquidGlass wires up for you.

import { createLiquidGlass } from "./src/liquid-glass.js";
createLiquidGlass(".glass"); // done
full effect ✨

Quickstart

Three lines. No bundler, no install — though npm and a CDN are there if you want them.

⬇️ Copy-paste (local files)

<!-- 1. styles -->
<link rel="stylesheet" href="src/liquid-glass.css">

<!-- 2. markup -->
<div class="glass">Hello</div>

<!-- 3. engine -->
<script type="module">
  import { createLiquidGlass } from "./src/liquid-glass.js";
  createLiquidGlass(".glass");
</script>

🌐 From the CDN (jsDelivr)

<link rel="stylesheet" href=
  "https://cdn.jsdelivr.net/gh/jbbixler/liquid-glass/src/liquid-glass.css">

<div class="glass">Hello</div>

<script type="module">
  import { createLiquidGlass, PRESETS }
    from "https://cdn.jsdelivr.net/gh/jbbixler/liquid-glass/src/liquid-glass.js";
  const g = createLiquidGlass(".glass");
  g.update(PRESETS.Frosted); // optional preset
</script>

Options — the six knobs

OptionDefaultWhat it does
scale34Edge magnification / refraction strength in px — higher bends more.
band0.9Width of the refractive ring as a multiple of the corner radius. Smaller = tighter to the edge.
blur2.5Frost amount (Gaussian blur, px).
sat1.8Saturation boost of the backdrop.
aberration0.17R/B channel split as a fraction of scale. 0 turns off the color fringe.
lens1+1 magnifies inward (default); -1 compresses the rim outward.

The handle exposes update(partial), rebuild(), destroy() and elements. Pass any preset object straight into update() — the same JSON the playground exports.
