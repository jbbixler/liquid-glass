Liquid Bubble

A discrete 3D glass droplet that magnifies and refracts the grid beneath it — chromatic aberration at the rim, a Fresnel edge glow, a moving specular hotspot. It free-drifts and springs to your cursor.

← Back to the Liquid Glass collection

Move your cursor over the grid — the droplet follows.

Your browser doesn’t support WebGL2, so the droplet can’t render. The image grid is shown without it. 

import { createLiquidBubble } from "./src/liquid-bubble.js";

createLiquidBubble({
  canvas,              // a <canvas> that fills the container
  container,           // a positioned box; the pointer is tracked over it
  cells,               // one DOM node per image, in imageUrls order
  imageUrls,           // CORS-clean URLs (same-origin or data:)
  onUnsupported: () => showFallback(),
});
